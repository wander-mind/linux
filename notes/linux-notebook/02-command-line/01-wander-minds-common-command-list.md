### wander-minds Common Command List:

| --*Basics*--  |**Description**                                            |
|---------------|-----------------------------------------------------------|
|date           |Print/Set system Date/Time                                 |
|cal            |Callendar                                                  |
|echo           |Display a line of text                                     |
|sort           |Sort lines of text files                                   |
|ls             |Lists directories                                          |
|cd             |Change directory                                           |
|pwd            |Print working directory                                    |
|type           |Type and location of program                               |
|which          |Find exact location of executable (not shell or aliases)   |
|touch          |Change file timestamp, but used to create empty files too  |
|file           |Determine file type                                        |
|mkdir          |Make a directory                                           |
|nano           |Text editor                                                |
|rm             |Remove file, *-r* remove folder, *-d* only empty folder    |
|mv             |Move (or rename) from-to (files and folders)               |
|cp             |Copy files and folder                                      |
|less           |Displays content of file one page at a time                |
|cat            |Concatenates and prints the contents of files              |
|tac            |*cat* in reverse                                           |
|rev            |Prints content of file line by line !!{tfel ot thgir}      |
|head           |Prints portion of file, by default from line 1-10          |
|tail           |Like *head*, but last 10 lines, not first 10               |
|wc             |Number of words, lines, or bytes in files                  |

| --*Manuals*-- |**Description**                                            |
|---------------|-----------------------------------------------------------|
|man            |Defacto Unix manuals                                       |
|info           |GNU manuals                                                |
|--help (-h)    |Short version of *man* pages                               |
|tldr           |5-10 most used options                                     |
